Shortcut Array Join

t's common knowledge that Array#*, when supplied with a number, multiplies the size of the array by using duplicate elements:

[1, 2, 3] * 3 == [1, 2, 3, 1, 2, 3, 1, 2, 3]

It's not well known, however, that when given a string as an argument Array#* does a join!

%w{this is a test} * ", "       # => "this, is, a, test"   h = { :name => "Fred", :age => 77 } h.map { |i| i * "=" } * "n" # => "age=77nname=Fred"
