Shortcut Array Join

t's common knowledge that Array#*, when supplied with a number, multiplies the size of the array by using duplicate elements:

[1, 2, 3] * 3 == [1, 2, 3, 1, 2, 3, 1, 2, 3]

It's not well known, however, that when given a string as an argument Array#* does a join!

%w{this is a test} * ", "       # => "this, is, a, test"

h = { :name => "Fred", :age => 77 }

h.map { |i| i * "=" } * "n"       # => "age=77nname=Fred"
Each

Calls the given block once for each element in self, passing that element as a paramete

a = [ "a", "b", "c" ]
a.each {|x| print x, " -- " }

produces:br a -- b -- c --
Map

Creates a new array containing the values returned by the block.

If no block is given, an Enumerator is returned instead.

a = [ "a", "b", "c", "d" ]
a.map! {|x| x + "!" }
a #=> [ "a!", "b!", "c!", "d!" ]
a.collect!.with_index {|x, i| x[0...i] }
a #=> ["", "b", "c!", "d!"]
